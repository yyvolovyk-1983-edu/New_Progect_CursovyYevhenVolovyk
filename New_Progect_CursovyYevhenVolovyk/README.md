# Курсовий проєкт

Цей репозиторій містить робочу структуру для підготовки курсової роботи. Основне завдання — оформити пояснювальну записку, зробити модель або симуляцію і підтвердити відповідність вимогам викладача.

## Що тут є

- `README.md` — коротка інструкція.
- `docs/project-plan.md` — план, етапи та графік.
- `docs/requirements.md` — вимоги та критерії.
- `docs/checklist.md` — контрольний список.
- `report/abstract.md` — анотація.
- `report/coursework_draft.md` — чернетка звіту.
- `src/main.py` — стартовий код.
- `src/simulation.py` — приклад симуляції.
- `docs/NEW_LAST_EDITION'S/` — робочі файли та схеми.
- `docs/матеріали для роботи/` — матеріали та джерела.
- `docs/Вимоги_до_КП/Структура курсової роботи.doc` — формальний шаблон.

## Як працювати

1. Визнач тему.
2. Заповни `docs/project-plan.md` і `docs/requirements.md`.
3. Перевір `docs/checklist.md`.
4. Напиши `report/abstract.md`.
5. Доповни `report/coursework_draft.md`.
6. Реалізуй модель у `src/`, якщо потрібно.

- Додай словник позначень.
- Використовуй `docs/NEW_LAST_EDITION'S/`.
