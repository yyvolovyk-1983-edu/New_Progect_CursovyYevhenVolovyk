# Структура курсового проекту

Ця схема показує, де зберігаються основні частини проекту.

## Каталоги

- `docs/` — документація, план, вимоги, робочі матеріали.
  - `docs/project-plan.md` — план проєкту, етапи та графік.
  - `docs/requirements.md` — вимоги до оформлення, критерії оцінки.
  - `docs/матеріали для роботи/` — наукові статті та джерела для моделювання.
  - `docs/NEW_LAST_EDITION'S/` — робочі файли, чернетки, ілюстрації та коментарі викладача.
  - `docs/Вимоги_до_КП/` — формальні технічні умови до курсової.

- `report/` — пояснювальна записка та додаткові тексти.
  - `report/abstract.md` — анотація.
  - `report/coursework_draft.md` — чернетка основної записки.

- `src/` — кодова частина.
  - `src/main.py` — стартовий приклад для реалізації.

## Поради

- Використовуй `docs/матеріали для роботи/` як джерело літератури і посилань.
- Всі основні розділи курсової виконай у `report/coursework_draft.md`, а потім збережи фінал у форматі PDF.
- Перевіряй вимоги до оформлення у `docs/requirements.md` під час підготовки звіту.
