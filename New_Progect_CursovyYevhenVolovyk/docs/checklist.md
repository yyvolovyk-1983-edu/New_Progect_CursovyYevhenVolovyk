# Контрольний список для виконання курсового проєкту

## Основні завдання
- [ ] Визначено тему, предмет і курс у `docs/project-plan.md`.
- [ ] Сформовано чіткі мету, завдання та обмеження проєкту.
- [ ] Визначено критерії оцінки та успіху.
- [ ] Підготовлено оформлену версію `docs/requirements.md`.

## Модель і методологія
- [ ] Описано об’єкт управління та систему координат.
- [ ] Побудовано лінійну модель динаміки каналу крену.
- [ ] Сформовано станову модель із визначеними матрицями A, B, C, G.
- [ ] Визначено сенсорну підсистему (IMU) та шум вимірювань.
- [ ] Включено опис блоку насичення/обмежень актуатора.

## Звіт і оформлення
- [ ] Перевірено відповідність структури викладацькому шаблону.
- [ ] Усі формули відокремлені від тексту і мають читабельний вигляд.
- [ ] Рисунки мають підписи, єдиний шрифт і зрозумілі позначення.
- [ ] Додано сучасні джерела 2020+ у список літератури.
- [ ] Звіт підготовлено з урахуванням A4, Times New Roman 14 pt, інтервал 1.5, поля 30/10/20/20.

## Реалізація та перевірка
- [ ] У `src/` є робочий приклад моделювання каналу крену.
- [ ] Перевірено, що модель і результати симуляції узгоджуються з поясненням.
- [ ] Збережено проміжні версії у git або аналогічному контролі версій.
- [ ] Результати підготовлено для експорту у PDF/ODT.

## Додаткові кроки
- [ ] Створити коротку анотацію у `report/abstract.md`.
- [ ] Перевірити, що всі позначення використовуються однаково у всіх документах.
- [ ] Переглянути коментарі викладача у `docs/NEW_LAST_EDITION'S/` і відобразити їх у чернетці.
